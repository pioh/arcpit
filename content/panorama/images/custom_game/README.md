### Лого для публикации (Workshop / Arcade)

В репозитории хранится **векторный исходник**: `arcpit_logo.svg`.

Для публикации обычно нужен **PNG 512×512**:

- **Файл**: `arcpit_logo_512.png`
- **Путь**: `content/panorama/images/custom_game/arcpit_logo_512.png`

Экспортировать можно любым способом:

- Inkscape: `File → Export PNG Image…` (512×512)
- Figma: `Export → PNG` (512×512)
- Любой онлайн SVG→PNG конвертер

После экспорта PNG Dota Tools обычно сам перекомпилирует ресурсы при публикации.
